# Components

Components are grouped with Atomic Design vocabulary while keeping the blog UI simple.

- `atoms`: primitive UI pieces such as icons and formatted values.
- `molecules`: small reusable controls or linked UI blocks.
- `organisms`: larger page sections such as navigation, footer, and content lists.
- `templates`: page-level composition components shared by Astro routes.

Prefer moving shared behavior into existing components before adding another layer or abstraction.
